Présentation du SSD Samsung SM961 512 GO

Une petite présentation du SSD Samsung SM961 512 GO

Il dispose de 512 go d'espace libre avec une vitesse d'écriture séquentielle de 3200 MB/S et jusqu'à 1700 MB/S en taux de transfert de données.

L'interface du SSD est en M.2

La présentation du SSD est malheureusement courte mais comme préciser dans le nom de l'article, je vais donner mon avis sur le "cloud computing"

Ce que je pense du cloud computing

Pour commencer qu'est ce que le cloud computing ?

en gros, le cloud computing c’est de pouvoir utiliser des ressources informatiques sans les posséder.

Donc pour faire court, à partir d'un simple boitier on peux utiliser un ordinateur à distance.

Mon avis sur le cloud computing

Le cloud computing à des avantages et des inconvenients, je vais commencer par les avantages.

Les avantages:

- Le prix de location par mois est accessible par tout le monde, par exemple 30€ par mois.

- Les caractéristiques de la machine.

Les inconvénients

- Obligation d'avoir une connexion internet haut débit, pour éviter les temps de latence (fibre ou VDSL2)

- Si les serveurs de l'entreprise plante ou sont en maintenance, il est impossible d'accèder à sa machine

- Mon avis sur le cloud computing

Comme dit ci-dessus le cloud computing à des avantages et des inconvénients, comme le prix mensuel, l'obligation d'avoir une très bonne connexion internet.

Je pense, même si malgré le prix, que c'est largement mieux d'opter pour l'achat d'un ordinateur portable ou pc fixe, étant donner que même sans internet on à toujours accès à ces données personnel.

Celui ci nous appartient également, ce qui n'est pas le cas avec le cloud computing, vu que ce sont des ordinateurs que l'on loue, par abonnement mensuel ou annuel.

Donc pour conclure pour moi, il est mieux d'avoir son ordinateur portable ou ordinateur fixe, ou l'on à toujours accès a ces données personnel même si l'on à plus accès à internet.
